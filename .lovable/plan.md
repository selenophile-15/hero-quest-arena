## 1단계: UI 개선 (이번 응답)
1. **앨범식 간격 조정** - 정렬/필터 박스와 소분류 탭 사이 간격 확대
2. **앨범식 카드 뒤집기** - 클릭 시 플립 애니메이션으로 스탯/씨앗 표시 + 관리 버튼
3. **스탯 계산표 색상 수정** - 화이트 모드에서도 다크 모드와 동일한 박스 색상 유지, 채도/명도 조정

## 2단계: 리스트 저장 개선 (다음 응답)
4. **내보내기 → 리스트 저장하기** - 영웅+챔피언/영웅만/챔피언만 선택, 특정 대상 체크 저장

## 3단계: 영웅 폼 기능 (이후 응답들)
5. **기타 상세 스탯 항목** - 15개 추가 스탯 필드 + 계산 반영
6. **비교 토글 모드** - 변경 전후 스탯 비교 시각화
7. **저장/다른 이름으로 저장** - 덮어쓰기/새 영웅 저장 분리
